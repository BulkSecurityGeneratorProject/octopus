Octopus
=======

Developped By Jhipster Archetype.


L'IHM est développé en :
  YEOMAN, GRUNT, BOWER, ANGULARJS


Le Backend utilise :
  * MAVEN, SPRING, SPRING MVC REST, SPRING DATA JPA



Demo : http://octopus.aymen.eu.cloudbees.net/




How to deploy
  * run mvn spring-boot:run
